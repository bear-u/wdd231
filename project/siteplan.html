Individual Project Site Plan

Site Name

NovaForge Games

This name represents a modern, creative, and innovative game development studio. “Nova” symbolizes new ideas and bright creativity, while “Forge” represents building, crafting, and developing games.

Optional domain availability: novaforgegames.com (example only)

Site Purpose

The purpose of this website is to introduce NovaForge Games as an independent game development studio, showcase our current and upcoming game titles, and provide a professional way for players, collaborators, and potential partners to contact us. The site will serve as a hub for game information, studio updates, and community engagement.

Scenarios

  • “What games is NovaForge currently developing, and where can I see screenshots or trailers?”
  • “How can I contact the studio for collaboration, business inquiries, or community involvement?”

Color Scheme

Primary Color: #00AAFF

Used for: headings, buttons, accents, highlights.

Secondary Color: #111111

Used for: backgrounds, card sections, footer, and contrast areas.

These colors create a modern, tech‑focused aesthetic that fits a game development brand. The bright blue provides energy and visibility, while the dark background supports a sleek, immersive feel.

Typography

Headings: Poppins

Used for all titles, section headers, and navigation elements.

Body Text: Roboto

Used for paragraphs, descriptions, and general content for readability.

These two fonts complement each other well: Poppins is bold and modern, while Roboto is clean and easy to read.
